This a Maven/Java application that matches patterns

Requirements
- Maven and Java should be installed in your computer :)

Instructions how to use:
1. Clone this Git project to your computer
2. Cd to root directory and type "mvn clean install"
3. Cd to target directory and type "java -cp <NAME OF JAR FILE>.jar timo.PatternMatcher" e.g. "java -cp glob-1.0-SNAPSHOT.jar timo.PatternMatcher"
4. Follow the instructions
